Github部署

Github仓库用于部署hexo网页静态文件。

创建一个仓库，名为：username.github.io

这里username必须是github账户名

Hexo 配置

  1. 在本地创建一个文件夹，用来安装hexo，我的文件夹名称和github仓库名相同。然后在该文件夹下打开终端，输入命令进行hexo 安装。

    然后在文件夹下会生成一些文件：

    为防止乱码，需要将_config.yml文件，scaffolds文件夹下的三个文件编码格式改为utf-8.

  2. 创建一个git版本库，用于将修改的文件push到github.

  3. 修改_config.yml文件中的属性

  4. 生成静态页面

  5. 预览页面

    在浏览器中输入 http://localhost:4000 就可以预览啦。
